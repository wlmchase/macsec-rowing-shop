# Secure Rowing Shop

## Objective

This full-stack web application project aimed to develop an e-commerce site for rowing gear with an emphasis on security measures. The goal was to achieve the expectations in the guidelines.

### Skills Learned

- Fullstack web development
- Proficiency in planning and designing a web application
- Ability to implement python and javascript code with security practices in mind.
- Enhanced knowledge of application protocols and security vulnerabilities.
- Development of critical thinking and problem-solving skills in application security.

### Tools & Technologies Used

- FastAPI
- PostgreSQL
- React
- OWASP Top 10
- NIST guidelines
- MITRE ATT&CK

## Steps

- Research Agile and DevOps methodologies and choose one of them
- Research and choose a suitable tech stack for the project
- Investigate common threats and attack patterns on web applications
- Design & Implementation
- Testing and Revisioning
